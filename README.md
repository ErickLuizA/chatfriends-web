## :clipboard: Description

ChatFriends is a chat app built with Nextjs, React Native (expo bare workflow) and firebase.

## :building_construction: Technologies

- [Firebase](https://firebase.google.com)

- [Typescript](https://www.typescriptlang.org/)

- [Nextjs](https://nextjs.org)

  - [React-icons](https://react-icons.github.io/react-icons/)

- [React Native](https://reactnative.dev/)
  - [Expo](https://expo.io)
  - [React Navigation](https://reactnavigation.org/)
  - [React-native-config](https://www.npmjs.com/package/react-native-config)

## :rocket: Getting Started

## 1. Download the repository

```shell
 $ git clone https://github.com/ErickLuizA/ChatFriends.git

 // Signin to firebase using your google account and configure your project
 // Follow the docs https://firebase.google.com/docs/web/setup
```

## 2. Installation

```shell
# Go into the repository
$ cd chatfriends
```

### Web

```shell
# Go to the web folder
$ cd web

# Install dependencies
$ yarn install
```

### Mobile

```shell
# Go to the mobile folder
$ cd mobile

# Install dependencies
$ yarn install
```

## 3. Execution

### Web

```shell
# On the web folder
$ yarn dev
```

### Mobile

```shell
# On the mobile folder
$ yarn android or yarn ios
$ yarn start
```
