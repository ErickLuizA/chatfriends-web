## :clipboard: Description

chatfriends is a chat app built with Nextjs and firebase.

## :building_construction: Technologies

- [Firebase](https://firebase.google.com)

- [Typescript](https://www.typescriptlang.org/)

- [Nextjs](https://nextjs.org)

  - [React-icons](https://react-icons.github.io/react-icons/)


## :rocket: Getting Started

## 1. Download the repository

```shell
 $ git clone https://github.com/ErickLuizA/chatfriends-web.git

 or 

 $ gh repo clone ErickLuizA/chatfriends-web

 // Signin to firebase using your google account and configure your project
 // Follow the docs https://firebase.google.com/docs/web/setup
```

## 2. Installation

```shell
# Go into the repository
$ cd chatfriends-web

# Install dependencies
$ yarn install
```

## 3. Execution

```shell
$ yarn dev
```

> [Live Demo](https://chat-friends.vercel.app/)
